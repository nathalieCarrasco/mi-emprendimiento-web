HTML y CSS para principiantes

HTML y CSS son dos herramientas importantes para dar inicio en el mundo del desarrollo web. Por lo tanto, en este curso aprenderás todo soobre HTML y CSS, desde lo más básico hasta un nivel intermedio, logrando desarrollar tu primera página web, sin la necesidad de aplicar librerías externas para CSS.

Estilos con SASS.

SASS es una herramienta immportante hoy en día para todo desarrollador web, ya que permite crear estilos más ordenados, eficientes y con menos líneas de código. Por ello, en este curso aprenderás todo lo neceario para crear hojas de estilo desde la sintaxis de SCSS, implementando técnicas propias de SASS.
